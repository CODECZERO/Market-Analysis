# Architecture Overview

## Executive Summary

Project Avichal uses a **polyglot microservices architecture** with:
- **TypeScript** for API Gateway, Aggregator, Orchestrator (reliability + ecosystem)
- **Rust** for primary worker (50MB RAM, free-tier compatible)
- **Python** for fallback worker (heavy ML, LangChain integration)
- **React** for frontend dashboard

The standout feature is the **3-Tier Intelligence Engine** that reduces AI API costs by 90% through intelligent local filtering.

### Key Features

#### 1. Content Deduplication

```rust
pub fn generate_content_hash(source_url: &str, text: &str) -> String {
    let mut hasher = Sha256::new();
    hasher.update(source_url.as_bytes());
    hasher.update(b"|");
    hasher.update(text.as_bytes());
    hex::encode(hasher.finalize())
}
```

Every mention is fingerprinted. MongoDB upserts ensure no duplicates.

#### 2. Smart GC (Gemini-Powered Compression)

When Redis memory hits 20MB (free tier buffer):

1. Fetch oldest 50 raw records
2. **Compress with Gemini** into a JSON summary
3. Archive to MongoDB `historical_summaries` collection
4. Trim Redis list

This maintains analytical value while respecting free-tier limits.

#### 3. Spike Detection

```rust
let threshold = f64::max(10.0, historical_average * 2.0);
let is_spike = (current_count as f64) > threshold;
```

Compares current mention volume against rolling 20-point historical average.
